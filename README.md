# Sniffly-ETV

This is a fork of Jason Dove's [ErsatzTV](https://github.com/ErsatzTV/ErsatzTV) that I intend for teaching myself a bit of .NET. In doing this, I plan to add a couple features to the final version of ErsatzTV.

## Current Planned Features

1. **Alternate schedules within sequential scheduling**: Alternate schedules with YAML/sequential scheduling based on specified dates and date ranges, as well as particular days of the week.
2. **Current date searches**: The ability to query the current month, day, or day of the week within searches (e.g. a smart collection for commercials that additionally plays holiday commercials when the month is December.)
